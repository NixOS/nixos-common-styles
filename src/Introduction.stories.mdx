# Welcome to NixOS site styles

In [this repository][this-repo] you will find common styles that are being used
on different web sites in [NixOS project][nixos-org].

Few examples where this styles are being used:

- [nixos.org](https://github.com/NixOS/nixos-homepage)
- [search.nixos.org](https://github.com/NixOS/nixos-search)
- [status.nixos.org](https://github.com/NixOS/nixos-status)

This [storybook][storybook] helps you build and discover different UI
components in isolation. That makes it easy to develop hard-to-reach states.

Browse example stories now by navigating to them in the sidebar. View their
code in the `stories/` directory.
