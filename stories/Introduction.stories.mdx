# Common styles for NixOS related web sites.

In [this repository][this-repo] you will find common styles that are being used
on different web sites in [NixOS project][nixos-org].

Few examples where this styles are being used:

- [nixos.org](https://nixos.org) ([Github](https://github.com/NixOS/nixos-homepage))
- [summer.nixos.org](https://summer.nixos.org) ([Github](https://github.com/NixOS/nixos-summer))
- (in progress) [search.nixos.org](https://search.nixos.org) ([Github](https://github.com/NixOS/nixos-search))
- (in progress) [status.nixos.org](https://status.nixos.org) ([Github](https://github.com/NixOS/nixos-status))


This documentation will help you build and discover different UI components in
isolation. That makes it easy to develop new NixOS web sites.

Browse example stories now by navigating to them in the sidebar. View their
code in the `stories/` directory.
